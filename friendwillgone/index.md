# 一位即将离开的朋友

**For a friend.**
<!--more-->

现在看到他将要离开了，心里还是有一点不舍的，毕竟有了一年了，还是有点感情的。

{{< /admonition >}}

现在心里想对他说的一句话就是：

> 到了新的专业，要努力学习，不要等到再见面的时候让我瞧不起你！！！
